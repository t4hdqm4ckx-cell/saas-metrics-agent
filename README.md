# FlowSync SaaS Metrics Dashboard Agent

> A fully self-contained B2C SaaS analytics platform with synthetic data generation, interactive dark-mode dashboard, and executive reporting — no backend or API key required.

![Dashboard Preview](docs/assets/preview.png)

---

## What Is This?

**FlowSync Metrics Agent** simulates a real B2C SaaS business — a productivity app with three subscription tiers — and generates 24 months of realistic operational data including revenue, churn, user engagement, and unit economics. The data powers a polished, interactive HTML dashboard built for operator and executive audiences.

### Key Features
- 📊 **4-tab interactive dashboard** — Revenue, Customers, Engagement, Unit Economics
- 🎨 **Dark mode UI** — Deep navy with electric cyan, purple, and neon green accents
- 📈 **10+ dynamic charts** — Area, bar, waterfall, donut, cohort heatmap, funnel
- 🤖 **Python data agent** — Reproducible synthetic data with seasonality and cohort effects
- 📋 **Executive deck** — Board-ready presentation in `docs/executive_deck.html`
- 🔒 **Security & audit docs** — SECURITY.md, AUDIT.md, ERROR_HANDLING.md
- 🔄 **Version tracking** — Fully documented change history

## Quick Start

### View the Dashboard (no setup required)
```bash
git clone https://github.com/t4hdqm4ckx-cell/saas-metrics-agent.git
cd saas-metrics-agent
open dashboard/index.html
```

### Serve Locally (optional)
```bash
python3 -m http.server 8080
# Open http://localhost:8080/dashboard/
```

### View Executive Deck
```bash
open docs/executive_deck.html
# Use browser Print → Save as PDF to export
```

**Company**: FlowSync Inc. (fictional)

| Tier | Price | Features |

### Synthetic Data Characteristics
- **24 months** of monthly snapshots (Jun 2024 – May 2026)
- Realistic **seasonality** (Q4 peaks, summer dips)

## Tech Stack
- **Frontend**: Vanilla HTML5, CSS3 (custom properties), ES6 JavaScript
- **Charts**: [Chart.js 4.x](https://www.chartjs.org/) via CDN
- **Data**: JSON, generated by Python 3.10+
- **Agent**: Python 3.10+, numpy, json, dataclasses
- **No framework, no build step, no API keys**
